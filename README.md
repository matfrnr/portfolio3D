# Portfolio 3D Interactif 🚀

Bienvenue sur le dépôt de mon portfolio futuriste et immersif. Ce projet est une vitrine interactive en 3D conçue pour présenter mon parcours, mes compétences et mes expériences professionnelles de manière dynamique.

## 🌟 Fonctionnalités

- **Environnement 3D Immersif** : Intégration de scènes 3D interactives (modèles PC desktop, éléments spatiaux/futuristes) avec gestion de la caméra et des lumières.
- **Animations Fluides** : Transitions animées et effets de survol réactifs pour une expérience utilisateur moderne.
- **Formulaire de Contact Intégré** : Envoi direct de messages via une API de messagerie tierce sans rechargement de page.
- **Design Futuriste & Responsive** : Interface soignée s'adaptant aux différentes tailles d'écrans.

## 🛠️ Technologies utilisées

Ce projet a été développé en combinant la puissance de la 3D sur le web et des frameworks front-end modernes :

- **Core :** React & Vite (pour un environnement de développement ultra-rapide)
- **3D / Graphismes :** Three.js, React Three Fiber (R3F), @react-three/drei (helpers et contrôles 3D)
- **Animations :** Framer Motion
- **Gestion des mails :** @emailjs/browser
- **Styles :** PostCSS & CSS moderne
